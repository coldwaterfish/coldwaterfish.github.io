#### wechat-snippet

```js
JSON 片段：在{}中输入 page,pages,window,tabbar等属性关键字即可提示代码片段模板。
JavaScript 片段：输入 wx-page,wx-app,wx-request等wx-开头的关键字即可提示代码片段模板。
WXML 片段：直接输入组件名称，即可提示代码片段模板。如,输入 view,可自动生成 View 标签及其属性。
```
